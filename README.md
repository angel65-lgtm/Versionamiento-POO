# Obsidian Nexus — Proyecto Telemedicina

Proyecto académico desarrollado para la asignatura Proyecto Integrador I,
integrando Programación Orientada a Objetos y Bases de Datos.

El sistema permite gestionar usuarios, doctores, especialidades y consultas médicas.

## Tecnologías utilizadas
- Python
- MySQL
- Programación Orientada a Objetos
